## <ins>Description</ins>

A progression Style web based game that allows users to choose between fighting, or gathering, as they progress through the game. The more resources the higher tier the equipment you may purchase. The game is live on Heroku and is hosted on Github.

## <ins>Installation</ins>

Intall all dependencies by running:

`npm install`

Available Seeds:

`npm start`

`npm run develop`

`npm run seed`
